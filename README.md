# PageCall 데모 모바일 애플리케이션
본 데모 앱은 고객사의 모바일 PageCall 사용 체험 및 PageCall Mobile SDK 도입을 돕기 위해 만들어졌습니다.

## 개요

- [iOS 데모](./ios) : PageCall iOS SDK 활용한 데모 애플리케이션
- [Node.js 서버 데모](./server) : PageCall 데모 애플리케이션을 위한 API 서버
